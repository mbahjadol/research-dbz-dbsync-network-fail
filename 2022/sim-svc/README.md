# simulate insertion and updating sql server data

## Start background insert

    curl http://localhost:8000/sim-insert-bg

## Start background update

    curl http://localhost:8000/sim-update-bg

## Check status
    
    curl http://localhost:8000/status

## Stop insert

    curl http://localhost:8000/stop-insert-bg

## Stop update
    
    curl http://localhost:8000/stop-update-bg
